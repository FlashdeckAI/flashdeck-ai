# FlashDeck.AI

**AI-powered flashcards for GCSE, IGCSE and A Level students.**

FlashDeck.AI generates exam-board-aligned flashcards instantly using Claude AI — complete with SVG diagrams for Biology, Chemistry, Physics and more. Used by students in 140+ countries.

## Tech Stack (MVP)

- **Domain:** flashdeck.ai (registered at Name.com)
- **Auth (planned):** Supabase (email/password + Google OAuth)
- **AI (planned):** Anthropic Claude Haiku API
- **Payments (planned):** Stripe
